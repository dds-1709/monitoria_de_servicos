# Monitoria de Serviços

Site estático pronto para publicação no **GitHub Pages**.

## Link público
Depois de subir este repositório no GitHub e executar o workflow de Pages, o link ficará em:

- `https://<SEU_USUARIO>.github.io/<NOME_DO_REPOSITORIO>/`

Exemplo para este projeto (se o nome do repositório for `monitoria_de_servicos`):

- `https://<SEU_USUARIO>.github.io/monitoria_de_servicos/`

## Como publicar
1. Faça push para o GitHub.
2. Vá em **Settings → Pages** e deixe como **Source: GitHub Actions**.
3. Aguarde o workflow **Deploy static site to GitHub Pages** finalizar.
4. Abra o link em **Actions** (campo `page_url`) ou no menu **Pages**.
